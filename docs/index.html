Quantifying differentiation timing differences from bulk RNA-seq time course data

Overview

PCA-based method for comparing differentiation progression between conditions in bulk RNA-seq time course data.

Dataset. Data from Martinez et al. (2024): D21 (euploid control) and T21 (trisomy 21) iPSC-derived neural progenitors at Days 6, 10, and 17 (18 samples, 3 replicates per condition).

Motivation

Standard differential expression compares conditions at individual timepoints independently. This discards the temporal structure of differentiation. A sample that is transcriptionally “ahead” or “behind” will appear differentially expressed at every timepoint, but the underlying cause is a timing shift, not a condition-specific program.

This method reduces the full transcriptomic trajectory to a single scalar maturation score per sample, enabling:

  • Direct quantification of differentiation timing differences between conditions
  • Detection of whether an experimental condition accelerates or delays differentiation
  • Comparison across experiments that share a common reference time course

Workflow

Mathematical Framework

PCA

Given a centered expression matrix \(\mathbf{X}_c\) (genes \(\times\) samples):

\[ \mathbf{X}_c = \mathbf{U}\boldsymbol{\Sigma}\mathbf{V}^\top \]

Each sample’s coordinates in PC space:

\[ \mathbf{z}_i = \mathbf{V}^\top \!\left(\mathbf{x}_i - \bar{\mathbf{x}}\right) \]

Centroid Vector

The reference vector connects control centroids at the earliest and latest timepoints:

\[ \vec{v} = \bar{\mathbf{z}}_{t_{\text{late}}} - \bar{\mathbf{z}}_{t_{\text{early}}} \qquad\text{where}\quad \bar{\mathbf{z}}_t = \frac{1}{n_t}\sum_{i \in t}\mathbf{z}_i \]

Maturation Score

\[ s_i = \frac{ \left(\mathbf{z}_i - \bar{\mathbf{z}}_{t_{\text{early}}}\right) \cdot \vec{v} }{ \lVert\vec{v}\rVert^2 } \]

By construction, \(s \approx 0\) at the early control centroid and \(s \approx 1\) at the late control centroid.

Feature Selection: Likelihood Ratio Test

DESeq2 LRT on D21 controls: full model includes timepoint, reduced model is intercept-only.

Code 
ctrl_meta <- meta[meta$genotype == "D21", ]
ctrl_counts <- raw_counts[, ctrl_meta$sample_id]

dds <- DESeqDataSetFromMatrix(
  countData = ctrl_counts,
  colData   = ctrl_meta,
  design    = ~ timepoint
)
dds <- DESeq(dds, test = "LRT", reduced = ~ 1)
res <- results(dds)
res <- res[order(res$padj), ]

sig_genes <- rownames(res)[
  !is.na(res$padj) & res$padj < 0.05

## Overview

PCA-based method for comparing differentiation
progression between conditions in bulk RNA-seq time
course data.

![](assets/overview.svg){width="100%"}

**Dataset.** Data from
[Martinez et al. (2024)](https://doi.org/10.3389/fncel.2024.1341141):
D21 (euploid control) and T21 (trisomy 21)
iPSC-derived neural progenitors at Days 6, 10,
and 17 (18 samples, 3 replicates per condition).

## Motivation

Standard differential expression compares conditions
at individual timepoints independently. This discards
the temporal structure of differentiation. A sample
that is transcriptionally "ahead" or "behind" will
appear differentially expressed at every timepoint,
but the underlying cause is a timing shift, not a
condition-specific program.

This method reduces the full transcriptomic trajectory
to a single scalar **maturation score** per sample,
enabling:

- Direct quantification of differentiation timing
  differences between conditions
- Detection of whether an experimental condition
  accelerates or delays differentiation
- Comparison across experiments that share a common
  reference time course

## Mathematical Framework

### PCA

Given a centered expression matrix $\mathbf{X}_c$
(genes $\times$ samples):

$$
\mathbf{X}_c
= \mathbf{U}\boldsymbol{\Sigma}\mathbf{V}^\top
$$

Each sample's coordinates in PC space:

$$
\mathbf{z}_i
= \mathbf{V}^\top
  \!\left(\mathbf{x}_i - \bar{\mathbf{x}}\right)
$$

### Centroid Vector

The reference vector connects control centroids at
the earliest and latest timepoints:

$$
\vec{v}
= \bar{\mathbf{z}}_{t_{\text{late}}}
  - \bar{\mathbf{z}}_{t_{\text{early}}}
\qquad\text{where}\quad
\bar{\mathbf{z}}_t
= \frac{1}{n_t}\sum_{i \in t}\mathbf{z}_i
$$

### Maturation Score

$$
s_i
= \frac{
    \left(\mathbf{z}_i
      - \bar{\mathbf{z}}_{t_{\text{early}}}\right)
    \cdot \vec{v}
  }{
    \lVert\vec{v}\rVert^2
  }
$$

By construction, $s \approx 0$ at the early control
centroid and $s \approx 1$ at the late control
centroid.

### Feature Selection: Likelihood Ratio Test

DESeq2 LRT on D21 controls: full model includes
timepoint, reduced model is intercept-only.

```{r}
#| label: lrt
#| cache: true

ctrl_meta <- meta[meta$genotype == "D21", ]
ctrl_counts <- raw_counts[, ctrl_meta$sample_id]

dds <- DESeqDataSetFromMatrix(
  countData = ctrl_counts,
  colData   = ctrl_meta,
  design    = ~ timepoint
)
dds <- DESeq(dds, test = "LRT", reduced = ~ 1)
res <- results(dds)
res <- res[order(res$padj), ]

sig_genes <- rownames(res)[
  !is.na(res$padj) & res$padj < 0.05
]
n_top <- min(1000, length(sig_genes))
top_genes <- head(rownames(res), n_top)

cat(
  "Significant genes (padj < 0.05):",
  length(sig_genes), "\n",
  "Genes used for PCA:", n_top
)
```

### Temporally Variable Gene Heatmap

Z-scored VST expression of all LRT-significant genes
in D21 samples. Rows split into $k = 3$ clusters.

```{r}
#| label: fig-heatmap
#| fig-cap: >
#|   Z-scored expression of temporally variable genes
#|   across D21 control timepoints (k=3 row clusters,
#|   viridis palette, winsorized at +/- 2)
#| fig-height: 6
#| fig-width: 4

hm_mat <- as.matrix(
  vst_counts[sig_genes, ctrl_meta$sample_id]
)
hm_z <- t(scale(t(hm_mat)))
hm_z[hm_z >  2] <-  2
hm_z[hm_z < -2] <- -2

col_fun <- colorRamp2(
  seq(-2, 2, length.out = 100),
  viridis(100)
)

  hm_z,
  name                   = "Z-score",
  col                    = col_fun,
  cluster_columns        = FALSE,
  cluster_rows           = TRUE,
  clustering_method_rows = "ward.D2",
  row_km                 = 3,
  row_gap                = unit(2, "mm"),
  show_row_names         = FALSE,
  show_column_names      = FALSE,

  column_gap      = unit(3, "mm"),
```

### D21 Control PC Space

PCA on D21 controls using top 1000 LRT genes defines
the reference coordinate system.

```{r}
#| label: pca-compute

pca_input <- t(as.matrix(
  vst_counts[top_genes, ctrl_meta$sample_id]
))
pca_res <- prcomp(pca_input, scale. = TRUE,
                  center = TRUE)

var_pct <- round(
  summary(pca_res)$importance[2, 1:3] * 100, 1
)

all_input <- t(as.matrix(
  vst_counts[top_genes, meta$sample_id]
))
all_proj <- predict(pca_res, all_input)

pca_df <- data.frame(
  PC1       = all_proj[, 1],
  PC2       = all_proj[, 2],
  PC3       = all_proj[, 3],
  sample    = meta$sample_id,
  genotype  = meta$genotype,
  timepoint = paste0("Day ", meta$timepoint)
)

tp_num_all <- as.numeric(
  as.character(meta$timepoint)
)
ctrl_d6  <- meta$genotype == "D21" & tp_num_all == 6
ctrl_d10 <- meta$genotype == "D21" & tp_num_all == 10
ctrl_d17 <- meta$genotype == "D21" & tp_num_all == 17

centroid_d6  <- colMeans(all_proj[ctrl_d6,  1:3])
centroid_d10 <- colMeans(all_proj[ctrl_d10, 1:3])
centroid_d17 <- colMeans(all_proj[ctrl_d17, 1:3])

  )),
  yaxis = list(title = paste0(
    "PC2 (", var_pct[2], "%)"

  zaxis = list(title = paste0(
```

### Sample Projections onto Trajectory

Each sample is projected onto $\vec{v}$ and
normalized:

$$
s_i
= \frac{
    \left(\mathbf{z}_i
      - \bar{\mathbf{z}}_{\text{Day}\,6}\right)
    \cdot \vec{v}
  }{
    \lVert\vec{v}\rVert^2
  }
$$

```{r}
#| label: scores-compute

scores <- apply(all_proj[, 1:3], 1, function(z) {
  sum((z - centroid_d6) * ref_vec) / sum(ref_vec^2)
})

score_df <- data.frame(
  sample    = meta$sample_id,
  genotype  = meta$genotype,
  timepoint = paste0("Day ", meta$timepoint),
  score     = scores
)
score_df$timepoint <- factor(
  score_df$timepoint,
  levels = c("Day 6", "Day 10", "Day 17")
)
```
