Quantifying differentiation timing differences from bulk RNA-seq time course data

Overview

This workflow quantifies how differentiation timing differs between experimental conditions using bulk RNA-seq data. It identifies temporally variable genes via the likelihood ratio test, constructs a reference trajectory through PCA space using control samples, and projects all samples onto that trajectory to compute a scalar maturation score.

Dataset. Publicly available data from Martinez et al. (2024): D21 (euploid control) and T21 (trisomy 21) iPSC-derived neural progenitors sampled at Days 6, 10, and 17 of differentiation. 18 samples total (3 replicates per genotype per timepoint).

Workflow

Mathematical Framework

Negative Binomial GLM

DESeq2 models read counts with a negative binomial distribution, where each gene \(i\) in sample \(j\) follows:

\[ K_{ij} \sim \text{NB}\!\left(\mu_{ij},\; \alpha_i\right) \]

\[ \log_2(\mu_{ij}) = \sum_r \beta_r^{(i)}\, x_{jr} \]

where \(\alpha_i\) is the gene-specific dispersion and \(x_{jr}\) are design matrix entries.

Likelihood Ratio Test

The LRT compares a full model (including timepoint) against a reduced model (intercept only):

\[ D = -2\left[ \ell\!\left(\hat{\boldsymbol{\theta}}_{\text{reduced}}\right) - \ell\!\left(\hat{\boldsymbol{\theta}}_{\text{full}}\right) \right] \;\sim\; \chi^2_{df_{\text{full}} - df_{\text{reduced}}} \]

Genes with significant \(D\) statistics (after BH correction) change expression across timepoints.

PCA

Given a centered expression matrix \(\mathbf{X}_c\) (genes \(\times\) samples), PCA computes the singular value decomposition:

\[ \mathbf{X}_c = \mathbf{U}\boldsymbol{\Sigma}\mathbf{V}^\top \]

Each sample’s coordinates in PC space:

\[ \mathbf{z}_i = \mathbf{V}^\top \!\left(\mathbf{x}_i - \bar{\mathbf{x}}\right) \]

Centroid Vector Projection

The reference vector connects control centroids at the earliest and latest timepoints:

\[ \vec{v} = \bar{\mathbf{z}}_{t_{\text{late}}} - \bar{\mathbf{z}}_{t_{\text{early}}} \qquad\text{where}\quad \bar{\mathbf{z}}_t = \frac{1}{n_t}\sum_{i \in t}\mathbf{z}_i \]

Maturation Score

Each sample is projected onto the reference vector and normalized so the early centroid maps to 0 and the late centroid maps to 1:

\[ s_i = \frac{ \left(\mathbf{z}_i - \bar{\mathbf{z}}_{t_{\text{early}}}\right) \cdot \vec{v} }{ \lVert\vec{v}\rVert^2 } \]

Scores below 0 indicate less-differentiated-than-early control; scores above 1 indicate more-differentiated-than-late control.

Feature Selection: Likelihood Ratio Test

The LRT is run on D21 control samples only. The full model includes timepoint; the reduced model is intercept-only. Genes where timepoint significantly explains variance are temporally variable.

Code 
# Subset to D21 controls
ctrl_meta <- meta[meta$genotype == "D21", ]
ctrl_counts <- raw_counts[, ctrl_meta$sample_id]

# DESeq2 LRT
dds <- DESeqDataSetFromMatrix(
  countData = ctrl_counts,
  colData   = ctrl_meta,
  design    = ~ timepoint
)
dds <- DESeq(dds, test = "LRT", reduced = ~ 1)
res <- results(dds)
res <- res[order(res$padj), ]

# Significant genes
sig_genes <- rownames(res)[
  !is.na(res$padj) & res$padj < 0.05
]

cat(
  "Significant genes (padj < 0.05):",
  length(sig_genes), "\n",
  "Top 10:", paste(head(sig_genes, 10), collapse = ", ")
)

Reference Vector and Centroid Projection

The reference trajectory is defined by connecting the D21 Day 6 centroid to the D21 Day 17 centroid in PC space. Each centroid is the mean of the replicate coordinates:

\[ \bar{\mathbf{z}}_{\text{Day}\,6} = \frac{1}{3}\sum_{i=1}^{3}\mathbf{z}_i^{(\text{D21, D6})} \qquad \vec{v} = \bar{\mathbf{z}}_{\text{Day}\,17} - \bar{\mathbf{z}}_{\text{Day}\,6} \]

Maturation Scores

Each sample is projected onto \(\vec{v}\) and normalized:

\[ s_i = \frac{ \left(\mathbf{z}_i - \bar{\mathbf{z}}_{\text{Day}\,6}\right) \cdot \vec{v} }{ \lVert\vec{v}\rVert^2 } \]

where \(s \approx 0\) at Day 6 and \(s \approx 1\) at Day 17 for D21 controls by construction.

Code 
# Compute scores
scores <- apply(all_proj[, 1:3], 1, function(z) {
  sum((z - centroid_d6) * ref_vec) / sum(ref_vec^2)
})

## Overview

This workflow quantifies how differentiation timing
differs between experimental conditions using bulk
RNA-seq data. It identifies temporally variable genes
via the likelihood ratio test, constructs a reference
trajectory through PCA space using control samples, and
projects all samples onto that trajectory to compute a
scalar maturation score.

![](assets/overview.svg){width="100%"}

**Dataset.** Publicly available data from
[Martinez et al. (2024)](https://doi.org/10.1016/j.stemcr.2024.04.010):
D21 (euploid control) and T21 (trisomy 21) iPSC-derived
neural progenitors sampled at Days 6, 10, and 17 of
differentiation. 18 samples total (3 replicates per
genotype per timepoint).

## Mathematical Framework

### Negative Binomial GLM

DESeq2 models read counts with a negative binomial
distribution, where each gene $i$ in sample $j$ follows:

$$
K_{ij} \sim \text{NB}\!\left(\mu_{ij},\; \alpha_i\right)
$$

$$
\log_2(\mu_{ij}) = \sum_r \beta_r^{(i)}\, x_{jr}
$$

where $\alpha_i$ is the gene-specific dispersion and
$x_{jr}$ are design matrix entries.

### Likelihood Ratio Test

The LRT compares a **full** model (including timepoint)
against a **reduced** model (intercept only):

$$
D = -2\left[
  \ell\!\left(\hat{\boldsymbol{\theta}}_{\text{reduced}}\right)
  - \ell\!\left(\hat{\boldsymbol{\theta}}_{\text{full}}\right)
\right]
\;\sim\; \chi^2_{df_{\text{full}} - df_{\text{reduced}}}
$$

Genes with significant $D$ statistics (after BH
correction) change expression across timepoints.

### PCA

Given a centered expression matrix $\mathbf{X}_c$
(genes $\times$ samples), PCA computes the singular
value decomposition:

$$
\mathbf{X}_c = \mathbf{U}\boldsymbol{\Sigma}\mathbf{V}^\top
$$

Each sample's coordinates in PC space:

$$
\mathbf{z}_i
= \mathbf{V}^\top
  \!\left(\mathbf{x}_i - \bar{\mathbf{x}}\right)
$$

### Centroid Vector Projection

The reference vector connects control centroids at the
earliest and latest timepoints:

$$
\vec{v}
= \bar{\mathbf{z}}_{t_{\text{late}}}
  - \bar{\mathbf{z}}_{t_{\text{early}}}
\qquad\text{where}\quad
\bar{\mathbf{z}}_t
= \frac{1}{n_t}\sum_{i \in t}\mathbf{z}_i
$$

### Maturation Score

Each sample is projected onto the reference vector and
normalized so the early centroid maps to 0 and the late
centroid maps to 1:

$$
s_i
= \frac{
    \left(\mathbf{z}_i
      - \bar{\mathbf{z}}_{t_{\text{early}}}\right)
    \cdot \vec{v}
  }{
    \lVert\vec{v}\rVert^2
  }
$$

Scores below 0 indicate less-differentiated-than-early
control; scores above 1 indicate
more-differentiated-than-late control.

### Feature Selection: Likelihood Ratio Test

The LRT is run on D21 control samples only. The full
model includes timepoint; the reduced model is
intercept-only. Genes where timepoint significantly
explains variance are temporally variable.

```{r}
#| label: lrt
#| cache: true

# Subset to D21 controls
ctrl_meta <- meta[meta$genotype == "D21", ]
ctrl_counts <- raw_counts[, ctrl_meta$sample_id]

# DESeq2 LRT
dds <- DESeqDataSetFromMatrix(
  countData = ctrl_counts,
  colData   = ctrl_meta,
  design    = ~ timepoint
)
dds <- DESeq(dds, test = "LRT", reduced = ~ 1)
res <- results(dds)
res <- res[order(res$padj), ]

# Significant genes
sig_genes <- rownames(res)[
  !is.na(res$padj) & res$padj < 0.05
]

cat(
  "Significant genes (padj < 0.05):",
  length(sig_genes), "\n",
  "Top 10:", paste(head(sig_genes, 10), collapse = ", ")
)
```

### Reference Vector and Centroid Projection

The reference trajectory is defined by connecting the
D21 Day 6 centroid to the D21 Day 17 centroid in PC
space. Each centroid is the mean of the replicate
coordinates:

$$
\bar{\mathbf{z}}_{\text{Day}\,6}
= \frac{1}{3}\sum_{i=1}^{3}\mathbf{z}_i^{(\text{D21, D6})}
\qquad
\vec{v}
= \bar{\mathbf{z}}_{\text{Day}\,17}
  - \bar{\mathbf{z}}_{\text{Day}\,6}
$$

```{r}
#| label: fig-centroid-3d
#| fig-cap: >
#|   3D PCA with reference vector connecting
#|   D21 Day 6 and Day 17 centroids

# Centroids
tp_num_all <- as.numeric(as.character(meta$timepoint))
d6_idx  <- meta$genotype == "D21" & tp_num_all == 6
d17_idx <- meta$genotype == "D21" & tp_num_all == 17

centroid_d6  <- colMeans(all_proj[d6_idx,  1:3])
centroid_d17 <- colMeans(all_proj[d17_idx, 1:3])
ref_vec <- centroid_d17 - centroid_d6

cat(
  "Day 6 centroid: ",
  paste(round(centroid_d6, 2), collapse = ", "), "\n",
  "Day 17 centroid:",
  paste(round(centroid_d17, 2), collapse = ", "), "\n",
  "Vector magnitude:",
  round(sqrt(sum(ref_vec^2)), 2)
)
```

### Maturation Scores

Each sample is projected onto $\vec{v}$ and normalized:

$$
s_i
= \frac{
    \left(\mathbf{z}_i
      - \bar{\mathbf{z}}_{\text{Day}\,6}\right)
    \cdot \vec{v}
  }{
    \lVert\vec{v}\rVert^2
  }
$$

where $s \approx 0$ at Day 6 and $s \approx 1$ at
Day 17 for D21 controls by construction.

```{r}
#| label: scores

# Compute scores
scores <- apply(all_proj[, 1:3], 1, function(z) {
  sum((z - centroid_d6) * ref_vec) / sum(ref_vec^2)
})

score_df <- data.frame(
  sample     = meta$sample_id,
  genotype   = meta$genotype,
  timepoint  = paste0("Day ", meta$timepoint),
  score      = scores
)
score_df$timepoint <- factor(
  score_df$timepoint,
  levels = c("Day 6", "Day 10", "Day 17")
)

# Summary table
score_summary <- score_df %>%
  group_by(genotype, timepoint) %>%
  summarize(
    mean_score = round(mean(score), 3),
    sd         = round(sd(score), 3),
    .groups    = "drop"
  )

knitr::kable(
  score_summary,
  col.names = c(
    "Genotype", "Timepoint", "Mean Score", "SD"

  caption = "Maturation scores by condition"
)
```

Positive differences indicate T21 is further along the
control differentiation trajectory; negative differences
indicate T21 is behind. The results suggest T21 neural
progenitors differentiate faster early but fail to reach
